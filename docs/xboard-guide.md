# Xboard 安装使用指南

## 1. 环境要求

- Docker 和 Docker Compose 环境
- 可访问的服务器（支持x86_64/ARM64架构）
- 开放端口：7001（默认Web端口）

## 2. 安装部署

### 2.1 获取项目代码

```bash
git clone https://github.com/cedar2025/Xboard.git
cd Xboard
```

### 2.2 配置环境

1. 复制环境配置文件：
```bash
cp .env.example .env
```

2. 修改.env文件中的必要配置：
- 数据库配置
- Redis配置
- 站点URL配置
- 邮件服务配置

### 2.3 使用Docker Compose部署

1. 复制compose配置文件：
```bash
cp compose.sample.yaml docker-compose.yml
```

2. 启动服务：
```bash
docker-compose up -d
```
